Service Account Key Discovery Critical

If a Firebase Service Account key is leaked (via exposed .json files, API endpoints, or misconfigured storage), an attacker can create Custom Auth Tokens for ANY user including admins. This bypasses all permission checks.

How it works: Service Account private key + firebase-admin SDK = ability to mint Custom Tokens with arbitrary UID. These tokens are then exchanged for real ID tokens via signInWithCustomToken. The resulting ID token has full authentication as the specified user.
